## üêõ Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| **Backend fails to start** | Check `MONGO_CONNECTION_STRING` in environment variables |
| **"Database connection failed"** | Verify MongoDB Atlas allows 0.0.0.0/0 access |
| **GitHub OAuth fails** | Verify callback URL in GitHub app settings matches backend URL |
| **CORS errors in browser** | Check `FRONTEND_URL` is set in backend environment |
| **Frontend can't reach backend** | Verify `VITE_API_BASE_URL` matches backend URL |
| **Cold starts (slow first request)** | Upgrade to Starter plan ($7/month) for always-on |
